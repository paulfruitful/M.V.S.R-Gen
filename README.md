# M.V.S.R-Gen
A generator based on the Python random module to randomly generate Mastercards, SSN, Visacards and Recharge cards

Tested on:
Termux
Linux
Windows
Debian

Installation process for Termux:
pkg upgrade
pkg update
pkg install git
git clone  https://github.com/paulfruitful/M.V.S.R-Gen.git
cd M.V.S.R-Gen
python mvsr.py
